# DNS Go-Between IP blacklist
# Comma/semicolon/newline-separated IPs or CIDRs.
#
# Use only ONE list at a time (whitelist OR blacklist).
# If both files have active rules, whitelist takes precedence.
#
# Any matching address is denied.
#
# Examples:
# 203.0.113.10
# 198.51.100.0/24
# 10.10.10.5,10.10.10.6
